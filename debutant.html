⌨️ Raccourcis utiles dans Scratch

  • Ctrl + C : Copier un bloc
  • Ctrl + V : Coller un bloc
  • Ctrl + D : Dupliquer un bloc
  • Suppr : Supprimer un bloc
  • Ctrl + Z : Annuler une action
  • Clic droit sur un bloc : Dupliquer / Supprimer

Astuce : Tu peux dupliquer seulement une partie de ton script avec clic droit > Dupliquer puis supprimer ce que tu ne veux pas garder.
