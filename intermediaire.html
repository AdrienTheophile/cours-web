Bienvenue dans le niveau Intermédiaire 💻

Ici, tu vas créer tes premiers vrais programmes en JavaScript. On quitte Scratch pour coder avec du "vrai code" !

Tu apprendras à manipuler le DOM, créer des interactions, et faire apparaître des éléments dynamiquement.

🎮 Projets disponibles

  • 🧠 Jeu de Memory
  • 🎵 Jeu de Simon
  • ⭕❌ Morpion (Tic Tac Toe)

💡 Tu peux ouvrir chaque jeu, jouer, puis explorer le code source pour comprendre comment il fonctionne.

📚 Concepts JS utilisés

  • Variables – Pour stocker des informations comme les scores
  • Boucles – Pour répéter des actions
  • Conditions – Pour tester les clics et vérifier des victoires
  • Événements – Pour réagir aux actions de l’utilisateur (clics, touches)
  • DOM – Pour manipuler les éléments HTML
